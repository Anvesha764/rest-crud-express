RESTful CRUD App

A simple RESTful CRUD (Create, Read, Update, Delete) web application built using Node.js, Express, and EJS.
This project is for practicing RESTful routing and backend fundamentals.

Tech Stack
- Node.js
- Express.js
- EJS
- UUID
- Method-Override
- HTML
- CSS
- JavaScript
