# RESTful CRUD App

A simple RESTful CRUD (Create, Read, Update, Delete) application built using Express and EJS.  
This project was created to practice backend fundamentals and RESTful routing.

## Tech Stack

- Node.js
- Express.js
- EJS
- UUID
- Method-Override
- HTML, CSS, JavaScript
