# RESTful CRUD App (Express + EJS)

A simple RESTful CRUD (Create, Read, Update, Delete) application built using **Node.js**, **Express**, and **EJS**.  
This project demonstrates backend fundamentals, RESTful routing, server-side rendering, and basic frontend styling.
